# Public Launch Post Draft

I just released the first public version of **ConsentMirror369**.

It is a humane pressure-literacy and consent-awareness tool for people who want to recognize when a conversation, request, offer, or argument may be rushing them, guilting them, shaming them, isolating them, or making it harder to choose clearly.

The goal is simple:

> Pressure reduces choice. Clarity restores choice.

ConsentMirror369 is not a tool for controlling people. It is not a tool for diagnosing people. It is not a tool for calling everyone manipulative.

A pattern is a cue, not proof.

It helps you pause and ask:

- Do I feel free to say no?
- Am I being rushed?
- Is guilt replacing consent?
- Is belonging being made conditional?
- Can I ask questions without punishment?
- What would I choose if fear, guilt, urgency, or shame were removed?

The app currently includes:

- a phrase analyzer
- 17 Mirror Sentences
- real-world practice scenarios
- a guided response builder
- local-only reflection log
- JSON export/import
- duplicate-aware imports
- printable card deck
- safety and privacy docs

The project vow:

> Lantern, not weapon.

Use it to return to breath, body, truth, consent, and self.
