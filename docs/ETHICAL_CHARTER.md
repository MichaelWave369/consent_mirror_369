# Ethical Charter

ConsentMirror369 exists to support agency, dignity, reflection, and clear choice.

## Core vows

1. Do not use this project to pressure people.
2. Do not use this project to shame people.
3. Do not treat a pattern as proof of intent.
4. Use the framework to pause, reflect, and choose clearly.
5. Seek trusted support when safety is involved.

## Tone

This project should sound calm, warm, honest, humble, and non-accusing.

## Safety boundary

If someone may be in danger, do not rely on a script. Prioritize trusted support, local emergency resources, or qualified help.
