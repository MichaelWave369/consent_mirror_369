# App Flow

## Goal

Help a user move from pressure or confusion into calm reflection.

## v0.1 flow

1. User chooses a sample phrase.
2. App shows a possible pattern.
3. App shows a mirror question.
4. App shows a boundary response.
5. App shows a consent check.

## Future flow

1. User enters a phrase.
2. App suggests possible pressure categories.
3. User selects what feels relevant.
4. App asks a body check question.
5. App suggests mirror questions.
6. App helps draft a respectful boundary response.
7. Optional local-only reflection log.

## Privacy principle

Local first. No hidden tracking. No selling user reflection data.
