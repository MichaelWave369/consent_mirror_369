# Guided Response Builder

The guided response builder helps users create calm boundary language without starting from a blank page.

## Current template tones

- Gentle pause
- Clear pause
- Firm boundary
- Evidence request
- Care with boundary
- Exit line

## Design principle

The builder should support agency and safety. It should not teach people to win, shame, corner, threaten, or overpower others.

## Template structure

Templates live in `data/response_templates.json` and include:

- `id`
- `tone`
- `label`
- `template`

## Safety note

If a direct response could increase danger, the safest response may be no direct response. The user should prioritize trusted support, distance, and qualified help when needed.
