# Card Deck Visual Identity

ConsentMirror369's printable deck should feel calm, protective, and practical.

## Core phrase

> Lantern, not weapon.

## Deck back language

- ConsentMirror369
- Pressure reduces choice. Clarity restores choice.
- Notice · Name · Pause · Choose
- A pattern is a cue, not proof.

## Visual direction

- Warm paper background
- Soft gold and moss accents
- Rounded card shapes
- Gentle radial glow, not harsh contrast
- Dashed inner border to suggest reflection and softness
- CM369 sigil as a simple placeholder mark

## Tone

The deck should not feel like a weapon, interrogation tool, or fear system. It should feel like a pocket lantern: calm, grounding, and agency-restoring.

## Future design options

- Replace the placeholder CM369 sigil with a polished mirror-lantern icon.
- Add optional grayscale print mode.
- Add cut guides.
- Add deck box art.
- Add scenario cards and response-builder cards.
