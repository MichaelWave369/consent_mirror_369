# Card Deck Spec

Each card should help a person pause and return to choice.

## Card fields

- Card name
- Pressure phrase
- Possible pattern
- Body signal
- Mirror question
- Boundary response
- Reflection prompt
- Safety note

## Starter card example

# Love as Pressure

**Phrase:** If you loved me, you would...

**Possible pattern:** Care or love being used as a test.

**Mirror question:** Do I feel free to care and still say no?

**Boundary response:** I care, and I still need my choice to be respected.

**Reflection prompt:** What would care look like with consent included?

**Safety note:** If saying no may create danger, prioritize support and safety planning over direct confrontation.
