# ConsentMirror369 v1.0.1 Release Notes

**Purpose:** Zenodo pickup / citation metadata release

## Summary

ConsentMirror369 v1.0.1 is a small metadata release intended to help Zenodo archive the project and mint a DOI after the repository has been connected in Zenodo.

## Added

- `CITATION.cff` citation metadata
- `.zenodo.json` Zenodo metadata

## Changed

- Package version bumped from `1.0.0` to `1.0.1`

## No app behavior changes

This release does not change the app behavior from v1.0.0. It only adds citation and archival metadata.

## Safety reminder

ConsentMirror369 is a humane pressure-literacy and consent-awareness project. A pattern is a cue to slow down, not proof of intent.
