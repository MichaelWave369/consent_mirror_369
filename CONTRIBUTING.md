# Contributing

Thank you for helping improve ConsentMirror369.

## Contribution principles

- Keep the tone calm and non-accusing.
- Do not write content that teaches people to pressure others.
- Treat patterns as reflection cues, not proof of intent.
- Include safety notes when a direct response could increase risk.
- Prefer plain language over jargon.

## Good contributions

- More mirror sentence examples
- Clearer boundary responses
- Better accessibility
- Better mobile layout
- Local-first privacy improvements
- Printable card deck designs
- Scenario examples for family, faith, workplace, online, sales, and relationships

## Avoid

- Diagnostic labels
- Revenge scripts
- Fear-amplifying language
- Content that encourages confrontation when safety is uncertain

## Local setup

```bash
npm install
npm run dev
```

Before opening a pull request:

```bash
npm run validate:data
npm run build
```
