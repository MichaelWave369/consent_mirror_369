# Security and Safety Policy

ConsentMirror369 is a local-first pressure-literacy and consent-awareness project.

## Supported version

| Version | Supported |
|---|---|
| 1.0.x | Yes |

## Reporting security concerns

If you find a security issue, please open a GitHub issue with enough detail to reproduce the concern, unless doing so would expose private user data or create risk for others.

For sensitive issues, contact the maintainer directly through the contact information listed on the GitHub profile.

## Safety concerns

ConsentMirror369 should not be used to pressure, shame, coerce, threaten, manipulate, or overpower people.

If you notice content that appears to teach harmful persuasion, coercive behavior, stalking, intimidation, or retaliation, please report it as a safety concern.

## Privacy model

The app stores reflections in browser local storage. It does not send those reflections to a server by default. Local storage is not encrypted, so anyone with access to the same browser profile may be able to view saved entries.

## Emergency disclaimer

ConsentMirror369 is not an emergency service. If someone is in immediate danger, they should contact trusted support, local emergency services, or qualified professional help.
